📖 사용법

1. 녹화

제목 입력 → 녹화 시작 → 웹페이지 클릭 시 자동 캡처
정지 — 일시정지 (편집 가능)
재개 — 녹화 이어서 계속
완료 — 녹화 종료
취소 — 모든 캡처 삭제

2. 캡처 모드

클릭당 1장 — 클릭마다 새 스크린샷
화면당 1장 — 같은 페이지 클릭은 하나에 번호 추가

3. 순서 변경

썸네일을 드래그하여 순서 변경하거나 ↑↓ 버튼 사용

4. 편집

세부 편집 — 전체 스텝을 큰 화면에서 편집
✏️ 편집기 — 마커/도형/텍스트 추가 (모달)

  • 사각형·화살표·원형·텍스트·넘버링·블러·크롭
  • 🖱 선택 — 클릭 → 드래그 이동 / 핸들로 크기 조절 / Delete 삭제
  • Ctrl+Z — 되돌리기

5. 재녹화

썸네일의 🔄 → 웹페이지 캡처(여러 장) → 완료로 확정

6. 저장/내보내기

💾 다운로드 폴더에 JSON 저장 (덮어쓰기)
📂 저장된 매뉴얼 불러오기
Export — HTML / PPT 내보내기

7. 단축키

Ctrl+Shift+E — 녹화 시작/정지 토글
PageUp/Down — 세부편집에서 스텝 이동

※ 녹화 중에는 썸네일 편집/순서변경이 잠금됩니다.
※ 크롬 주소창·확장프로그램 UI는 캡처되지 않습니다.
