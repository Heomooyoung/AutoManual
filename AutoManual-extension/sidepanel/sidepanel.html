📖 사용법

1. 녹화

녹화 시작 버튼 클릭 후 웹페이지에서 원하는 곳을 클릭하면 자동으로 스크린샷이 캡처됩니다.

2. 캡처 모드

클릭당 1장 — 클릭할 때마다 새 스크린샷
화면당 1장 — 같은 페이지 클릭은 하나의 스크린샷에 번호 추가

3. 편집기 도구

각 단계의 편집 버튼을 클릭하면 편집기가 열립니다.

  • ▭ 사각형 / → 화살표 / ● 원형 — 드래그로 그리기
  • T 텍스트 — 클릭 후 직접 타이핑 (Enter 확정)
  • ① 넘버링 — 클릭하면 순번 원형 마커 배치
  • ◫ 블러 — 드래그로 모자이크 처리
  • ⬒ 크롭 — 상하좌우 핸들로 영역 조절
  • 🖱 선택 — 마커 클릭 후 Delete로 삭제
  • Ctrl+Z — 모든 작업 되돌리기

4. 내보내기

녹화 완료 후 HTML / PPT / GIF 형식으로 내보낼 수 있습니다.

※ 브라우저 주소창, 확장프로그램 UI 등 크롬 자체 영역은 캡처되지 않습니다.

클릭할 때마다 새 스크린샷을 캡처합니다
