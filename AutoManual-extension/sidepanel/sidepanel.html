📖 사용법

1. 녹화

제목 입력 → 녹화 시작 → 웹페이지 클릭 시 자동 캡처

2. 캡처 모드

클릭당 1장 — 클릭마다 새 스크린샷
화면당 1장 — 같은 페이지 클릭은 하나에 번호 추가

3. 순서 변경

썸네일을 드래그하여 순서 변경하거나 ↑↓ 버튼 사용

4. 편집기

✏️ 버튼으로 편집기 열기

  • 사각형·화살표·원형·텍스트·넘버링·블러·크롭
  • 🖱 선택 — 클릭 선택 → 드래그 이동 / Delete 삭제
  • Ctrl+Z — 되돌리기

5. 재녹화

🔄 클릭 → 웹페이지 캡처(여러 장 가능) → 완료로 확정

6. 저장/내보내기

💾 저장 / 📂 불러오기 / HTML·PPT·GIF 내보내기

※ 크롬 주소창·확장프로그램 UI는 캡처되지 않습니다.
